# How to run

## Option 1 — Plane walking
py scripts/run_dcm_walk.py to make the Robot Walk in the plain environment.

## Option 2 — Little Stairs
py scripts/run_dcm_walk.py --scene "scene_stairs.xml" to make the Robot Walk in the stairs environment.

## Option 3 — Tilted scene
py scripts/run_dcm_walk.py --scene "scene_tilted.xml" to make the Robot Walk in the tilted environment.
